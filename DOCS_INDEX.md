# CraftPath Docs and Data Tables Index

Generated by cataloging the CraftPath working tree. This index plus the files beside it are the documentation and the datamined data tables for the solver. Descriptions are by file name and category. For exact schema, read the loaders: item_parser.py, build_weights.py, desecrated.py, essences.py.

## Documentation

- ALLOY_NOTES.md: Notes on the 0.5 Alloy / crafted-mod interaction.
- BACKLOG.md: Open backlog and known gaps (per-base WEIGHTS pipeline is the top item).
- DEPLOY.md: Render deployment steps.
- DESKTOP_BUILD.md: PyInstaller desktop build instructions.
- DEV_NOTES.md: Developer notes and internals.
- GITHUB_SETUP.md: Git remote and repo setup.
- README.md: Project overview, features, and usage.
- RELEASE_NOTES.md: Release history.
- START_HERE.txt: End-user quickstart (online vs desktop editions).
- WEIGHTS_HOWTO.md: How the per-base mod weight pipeline works and how to regenerate it.

## Notes

- data/desecrated_raw.txt is a raw datamined source dump and contains long-dash characters from the source. It was left byte-for-byte unchanged here to avoid breaking whatever parses it. Everything else in this package is long-dash free.
- START_HERE.txt in this package had four separator long-dashes converted to hyphens to match the project rule. The copy on your disk was not modified.
